# FlowGrid MemoryAgent 中文旁白

记忆，不等于授权。

一个长期项目里，已经确认的决策，和刚刚提出的新要求，可以同时存在。

FlowGrid MemoryAgent 使用 Qwen 从对话中提取可持续使用的判断，但每一条新记忆一开始都只是待确认状态。

这里，Qwen 提取出了“本地优先”的项目方向。我授权它之后，它才成为 Agent 可以继承和执行的正式判断。

刷新页面后，这条确认过的决策仍然存在。

现在出现一条冲突的新要求：立刻把项目改成云端优先。Qwen 会把它记录为待确认的修订，而不会直接覆盖原来的正式决策。

当我询问现在什么是被授权执行的内容时，Agent 只会检索已经确认的判断，并说明这次修改仍然需要人工批准。

Qwen 负责推理。阿里云函数计算托管接口。CockroachDB 保存跨会话的状态。

FlowGrid MemoryAgent 保存的，不只是 Agent 听到过什么，而是它当前被允许依据什么行动。
